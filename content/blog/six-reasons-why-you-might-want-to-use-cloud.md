For the past few weeks, I’ve been trying to learn AWS. For those who are not in IT, AWS stands for Amazon Web Services. In simple terms, AWS provides services for people who want to perform any computing task in the cloud (as opposed to on-premises). 

## Cloud Computing

You are reading this on a computer, whether a handheld device or a PC. This site is also hosted on a computer, but instead of a personal device, it runs on a server. When people refer to cloud computing, they usually mean using a server. A server, as the name suggests, serves the user with anything from TikTok videos, threads on Thread, cat videos that we all love, and much more. All the content that is accessible to us on the internet is actually being stored somewhere and being sent to the users when requested.
